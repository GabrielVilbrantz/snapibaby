# Skill: paywall-cro
## Depends on: product-marketing-context

## O que faz
Otimiza o momento e a forma como o paywall (preview -> pagamento) e apresentado.

## Modelo atual SnapiBaby
Preview gratis com marca d'agua -> Pagamento para remover marca d'agua e receber HD

## Opcoes de paywall
1. Hard gate: preview com marca d'agua grande (atual)
2. Soft gate: preview com marca d'agua discreta + email gate
3. Freemium: 1 foto gratis em baixa resolucao

## Hipotese de teste
Mostrar o preview com marca d'agua PRIMEIRO, depois mostrar preco
vs
Mostrar preco ANTES do preview

## Psicologia do paywall
- Efeito dotacao: usuario ja "possui" a foto mentalmente apos ver o preview
- Loss aversion: "Don't lose this portrait" > "Buy this portrait"
- Social proof: "18,247 moms unlocked their portraits"

## Output
Copy alternativo para a tela de paywall
A/B test setup
