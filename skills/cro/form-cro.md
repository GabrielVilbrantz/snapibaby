# Skill: form-cro
## Depends on: product-marketing-context

## O que faz
Otimiza formularios especificos — checkout, upload, qualquer form do site.

## Principios
1. Um campo por linha (nunca dois campos side-by-side no mobile)
2. Label acima do campo (nunca placeholder como unico label)
3. Tipo correto de input (tel para telefone, email para email)
4. Validacao em tempo real (nao so no submit)
5. Botao de submit: verbo de acao + beneficio ("Get My Portraits — ")
6. Microcopy abaixo do botao: remover ultima objecao
