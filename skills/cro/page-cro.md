# Skill: page-cro
## Depends on: product-marketing-context

## O que faz
Otimiza qualquer pagina especifica do SnapiBaby para maximizar conversao.
Versao especializada de position-me para uma pagina por vez.

## Protocolo
### Para a homepage (index.html)
1. Hero: proposta de valor em <=7 palavras acima do fold?
2. CTA primario: visivel sem scroll no mobile (375px viewport)?
3. Prova social: aparece antes do primeiro CTA?
4. Preco: aparece junto da proposta de valor ou so na secao de planos?
5. Objecoes: respondidas antes do botao de compra?
6. Exit intent: popup ativo?

### Para checkout (checkout.html)
1. Numero de campos: <=3 antes do cartao?
2. Order bump: claro e com preco de urgencia?
3. Trust signals: SSL badge, garantia, reviews?
4. Abandono: existe recuperacao por email?

## Benchmarks
- Hero CTA click rate: >5%
- Checkout completion rate: >60%
- Order bump attach rate: >25%

## Output
Lista priorizada de mudancas com impacto estimado em CVR
Codigo HTML/CSS do fix pronto para implementar
