# Skill: revops
## Depends on: product-marketing-context

## O que faz
Otimiza as operacoes de receita do SnapiBaby — funil completo de aquisicao a retencao.

## Metricas do funil atual (estimativas)
Visitante -> CTA: 35%
CTA -> Upload: 60%
Upload -> Preview: 85%
Preview -> Checkout: 40%
Checkout -> Pagamento: 65%
Pagamento -> Upsell aceito: 25%

## CVR geral estimada: visitante -> pagamento = ~5%
Benchmark SaaS: 2-5% | E-commerce: 1-3% | SnapiBaby alvo: >6%

## Maior alavanca atual
Preview -> Checkout (40%) = maior drop. Cada ponto percentual = +2.5% de receita total.

## Stack de revops necessario
- CRM: Supabase (ja tem) para historico de clientes
- Email: MailerLite ou Brevo (gratuito ate 9k emails/mes)
- Analytics: GA4 (instalar)
- Heatmap: Microsoft Clarity (gratis)
- A/B: implementacao manual com JS (sem custo)

## Output
Dashboard de metricas-chave para monitorar semanalmente
