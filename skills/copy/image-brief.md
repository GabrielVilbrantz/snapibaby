# Skill: image-brief
## Depends on: product-marketing-context

## O que faz
Cria briefs de imagem para designers, photographers ou gerador de imagens AI para assets do SnapiBaby.

## Tipos de imagem necessarios
1. Hero mockup: celular mostrando before/after (fundo branco removido)
2. Theme previews: retrato AI em cada um dos 15 temas
3. Ad creatives: verticais (9:16) para Meta/TikTok
4. Blog headers: 1200x630px por post
5. Social assets: Instagram square + stories

## Brief template
Tipo: [hero/ad/blog/social]
Dimensoes: [px]
Conteudo: [descricao do que deve aparecer]
Tom: [emocional/informativo/urgencia]
CTA visual: [elemento de texto na imagem]
Cores: Rosa SnapiBaby (#f43f75) + Branco + Creme

## Output
Brief completo pronto para passar para designer ou Midjourney/DALL-E
