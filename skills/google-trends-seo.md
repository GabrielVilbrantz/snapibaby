# Skill: Google Trends SEO

## Quando usar
Mencione "Google Trends SEO" para ativar análise de keywords com timing e sazonalidade.

## O que esta skill faz
Cruza as keywords do SnapiBaby com tendências do Google Trends para identificar:
- Quando publicar cada conteúdo (timing de sazonalidade)
- Quais keywords estão em ascensão vs declínio
- Oportunidades de conteúdo trending antes da concorrência

---

## Protocolo de Execução

### Passo 1 — Verificar Google Trends
Acessar via browser: `https://trends.google.com/trends/explore?q=KEYWORD&geo=US`

Keywords a verificar para SnapiBaby:

**Tier 1 — Monitorar mensalmente:**
- `newborn photography`
- `AI newborn photos`
- `newborn photoshoot`
- `newborn photographer cost`

**Tier 2 — Verificar trimestralmente:**
- `baby photo ideas`
- `newborn photo props`
- `newborn studio photography`
- `at home newborn photoshoot`

**Tier 3 — Sazonalidade:**
- `newborn christmas photos` (pico: Nov-Dez)
- `newborn easter photos` (pico: Mar-Abr)
- `baby valentine photos` (pico: Jan-Fev)
- `newborn halloween costume` (pico: Set-Out)

### Passo 2 — Análise de sazonalidade

Padrões conhecidos para newborn photography:
| Mês | Tendência | Ação |
|---|---|---|
| Janeiro | Alta (bebês de outubro/novembro) | Publicar conteúdo de custo/comparação |
| Fevereiro | Alta + Valentine's | Publicar tema Valentine/Love |
| Março | Média + Easter prep | Publicar tema Easter Bunny |
| Abril | Easter peak | CTA urgência "limited time Easter theme" |
| Maio | Alta (Mother's Day) | Gift guide para novas mães |
| Junho | Média | Blog educacional |
| Julho | Baixa | Preparar conteúdo de outono |
| Agosto | Subindo | Back to school → newborn content |
| Setembro | Alta | Bebês de inverno chegando |
| Outubro | Alta + Halloween | Tema Halloween/Pumpkin |
| Novembro | Pico anual | Black Friday promo + Christmas photos |
| Dezembro | Pico anual | Christmas theme, gift urgência |

### Passo 3 — Identificar keywords em ascensão

Verificar seção "Related queries" no Google Trends em modo "Rising" para:
- Novas variações de "AI baby photos"
- Termos emergentes em parenting tech
- Keywords geográficas crescendo (novas cidades a adicionar)

### Passo 4 — Output

```
## Google Trends SEO Report — [Data]

### Keywords em alta agora
[Keyword] — [% crescimento] — [Ação recomendada]

### Oportunidade sazonal próxima (próximos 30 dias)
[Evento] → [Keywords] → [Conteúdo a criar]

### Páginas a criar/atualizar agora
1. [URL] — [Keyword] — [Timing]

### Keywords declining (desinvestir)
[Keyword] — [Tendência] — [Redirecionar esforço para]
```

### Passo 5 — Implementar
Criar ou atualizar blog posts e páginas de tema com base nas oportunidades identificadas.
Adicionar novos posts ao `blog-post.html` (objeto POSTS).
